### Типы и семантика
- [x] Запрещено присваивание значения типа float переменной типа int.
- [x] Запрещено объявление переменной с указанием типа int и инициализацией значением типа float.
- [x] Запрещено объявление переменной типа int с инициализацией строковым значением.
- [x] Запрещено объявление вложенных функций.
- [x] Функция с объявленным типом возвращаемого значения должна корректно возвращать значение этого типа.
- [x] Функция без указания типа возвращаемого значения может завершаться без оператора return.
- [x] Локальная переменная функции недоступна вне тела функции.
- [x] Запрещено сокрытие параметра функции локальной переменной с тем же именем.
- [x] Параметры функции передаются по значению (изменение параметра не изменяет аргумент).
- [x] Несовпадение количества фактических и формальных параметров приводит к ошибке.
- [x] Фактические параметры функции вычисляются слева направо.
- [x] Поддерживается прямая рекурсия, но взаимная рекурсия запрещена.
- [x] В условии ветвления значение 0 считается ложным, ненулевое — истинным.
- [x] Условие цикла while является целочисленным выражением, инструкции break и continue работают корректно.
