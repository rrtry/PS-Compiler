# Сравнение синтаксиса выражений: **Python vs C#**

---

## 1. Сравнение числовых типов данных

### **Python**
```python
# Целые числа (int — произвольной длинны)
x = 42
big_num = 123456789012345678901234567890

# Вещественные (float)
y = 3.14
z = 2.5e-3

# Комплексные (complex)
c = 3 + 4j

# Логический (bool — подтип int)
flag = True  # Эквивалентно 1
```

### **C#**
```csharp
// Целые числа со знаком
sbyte a = 127;    // 8-bit
short b = 32000;  // 16-bit
int c = 42;       // 32-bit
long d = 100L;    // 64-bit

// Целые числа без знака
byte e = 255;
ushort f = 65000;
uint g = 4000000000U;
ulong h = 100UL;

// Вещественные
float i = 3.14f;     // 32-bit
double j = 2.718;    // 64-bit
decimal k = 123.45m; // 128-bit (для финансовых расчетов)

// Логический
bool flag = true;
```

**Ключевые различия:**
- Python использует **динамическую типизацию**, C# — **статическую**.  
- В Python `int` имеет **произвольную точность**, в C# — фиксированные размеры.  
- В C# есть отдельный тип **`decimal`** для точных финансовых расчетов.  
- Python поддерживает **комплексные числа** на уровне языка.

## 2. Сравнение арифметических операторов

### **Общие операторы (одинаковые в обоих языках)**
```python
a + b    # Сложение
a - b    # Вычитание
a * b    # Умножение
a / b    # Деление
a % b    # Остаток от деления
-a       # Унарный минус
+a       # Унарный плюс
```

### **Уникальные для Python**
```python
a ** b      # Возведение в степень
a // b      # Целочисленное деление
```

### **Уникальные для C#**
```csharp
a++         // Инкремент (постфиксный)
++a         // Инкремент (префиксный)
a--         // Декремент (постфиксный)
--a         // Декремент (префиксный)
```

**Различия:**
- В Python деление `/` **всегда возвращает float**, даже для целых.  
- В C# при делении двух `int` результат также `int` (отбрасывается дробная часть).  
- Python не имеет операторов `++` и `--`.

## 3. Сравнение приоритетов операций

### **Python (от высшего к низшему):**
```
**                  # Возведение в степень (правоассоциативное)
+x, -x, ~x          # Унарные
*, /, //, %         # Умножение и деление
+, -                # Сложение и вычитание
<<, >>              # Битовые сдвиги
&                   # Побитовое И
^                   # Побитовое исключающее ИЛИ
|                   # Побитовое ИЛИ
```

### **C# (от высшего к низшему):**
```
x++, x--            # Постфиксные
+x, -x, !, ~, ++x, --x   # Префиксные
*, /, %             # Умножение и деление
+, -                # Сложение и вычитание
<<, >>              # Битовые сдвиги
<, >, <=, >=        # Сравнения
==, !=              # Равенство/неравенство
&                   # Побитовое И
^                   # Исключающее ИЛИ
|                   # Побитовое ИЛИ
```

**Заметка:**  
Приоритеты в C# **более детализированы** (включают сравнения и логические операции).  
В Python приоритет проще, но `**` имеет **уникальную правоассоциативность**.

## 4. Список лево- и правоассоциативных операторов

### **Python**
**Правоассоциативные:**
- `**` (возведение в степень)

**Левоассоциативные:**
- Все остальные арифметические (`+`, `-`, `*`, `/`, `//`, `%`)  
- Битовые (`<<`, `>>`, `&`, `|`, `^`)  

---

### **C#**
**Правоассоциативные:**
- `=`, `+=`, `-=`, `*=`, `/=`, `%=` (операторы присваивания)  
- `??` (null-coalescing)  

**Левоассоциативные:**
- Все арифметические (`+`, `-`, `*`, `/`, `%`)  
- Сравнения (`==`, `!=`, `<`, `>`)  
- Битовые (`&`, `|`, `^`)

## 5. Сравнение встроенных функций для работы с числами

### **Python**
```python
# Математические функции (модуль math)
abs(x)              # Абсолютное значение
pow(x, y)           # Возведение в степень
divmod(a, b)        # Возвращает (a // b, a % b)
round(x[, n])       # Округление

# Преобразования типов
int("123")          # Преобразование в int
float("3.14")       # Преобразование в float
complex(3, 4)       # Создание комплексного числа

# Проверки типов
isinstance(x, int)  # Проверка типа
type(x)             # Получение типа
```

### **C#**
```csharp
// Статические методы классов
Math.Abs(x);         // Абсолютное значение
Math.Pow(x, y);      // Возведение в степень
Math.Round(x);       // Округление
Math.Max(a, b);      // Максимум
Math.Min(a, b);      // Минимум

// Преобразования типов
int.Parse("123");
Convert.ToInt32("123");
double.Parse("3.14");
x.ToString();        // В строку

// Проверки типов
x is int;
x.GetType();
```

**Вывод:**  
Python использует **функции верхнего уровня** и модули (`math`, `cmath`),  
C# — **статические методы** из классов `Math`, `Convert`, `System.Numerics`.

## 6. Прочие особенности синтаксиса выражений

### **Автоматическое приведение типов**
**Python:**
```python
# Динамическое приведение в выражениях
result = 5 + 3.14    # int + float → float
result = True + 42   # bool + int → int (True = 1)
```

**C#:**
```csharp
// Статическое приведение требуется явно
double result = 5 + 3.14;        // Неявное расширение int → double
int a = (int)3.14;               // Явное сужающее приведение
decimal b = (decimal)3.14;       // Явное приведение
```

---

### **Перегрузка операторов**

**Python:**
```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    
    def __add__(self, other):
        return Vector(self.x + other.x, self.y + other.y)
```

**C#:**
```csharp
public struct Vector
{
    public double X, Y;
    
    public Vector(double x, double y) { X = x; Y = y; }
    
    public static Vector operator +(Vector a, Vector b)
    {
        return new Vector(a.X + b.X, a.Y + b.Y);
    }
}
```


### **Дополнительно:**
- Python позволяет **цепные сравнения**:  
  ```python
  if 0 < x < 10:
      print("x in range")
  ```
  В C# это пишется как:
  ```csharp
  if (x > 0 && x < 10)
      Console.WriteLine("x in range");
  ```

- В Python **числа не переполняются**, в C# возможны **исключения OverflowException**.  
- Python допускает **параллельное присваивание** (`x, y = y, x`),  
  C# — только через кортежи (`(x, y) = (y, x);`).
