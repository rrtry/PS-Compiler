# Лексическая структура языков Python и C#

## 1. Общие сведения
Лексическая структура определяет базовые элементы синтаксиса языка программирования: идентификаторы, ключевые слова, литералы, операторы, разделители и правила формирования кода.

---

## 2. Блоки кода

| Элемент | Python | C# |
|----------|---------|----|
| Обозначение блока | Отступы (обычно 4 пробела) | Фигурные скобки `{ }` |
| Область видимости | Определяется уровнем отступа | Определяется скобками |

Пример
 ```python
if x > 0:
    print(x)
```

```csharp
if (x > 0) 
{
    Console.WriteLine(x);
}
```

**Различие:** Python использует структурную индентацию вместо скобок, что делает форматирование синтаксически значимым.

---

## 3. Ветвления

| Элемент | Python | C# |
|----------|---------|----|
| Ключевые слова | `if`, `elif`, `else` | `if`, `else if`, `else` |
| Условия | Без скобок, логические выражения | В круглых скобках |

Пример
 ```python
if a > b:
    print(a)
elif a == b:
    print('=')
else:
    print(b)
``` 

```csharp
if (a > b) 
{
    Console.WriteLine(a);
} 
else if (a == b) 
{
    Console.WriteLine("=");
} 
else 
{
    Console.WriteLine(b);
}
```

**Различие:** C# требует скобки вокруг условий и фигурные скобки вокруг блоков; Python — нет.

---

## 4. Циклы

| Тип цикла | Python | C# |
|------------|---------|----|
| `for` | Итерация по коллекции или диапазону | Счетчик с условием |
| `while` | Пока условие истинно | Пока условие истинно |
| Прерывания | `break`, `continue` | `break`, `continue` |

Пример
 ```python
for i in range(5):
    print(i)
``` 

```csharp
for (int i = 0; i < 5; i++) 
{
    Console.WriteLine(i);
}
```

**Различие:** В Python цикл `for` итерационный (по объектам), в C# — счетный.

---

## 5. Ввод и вывод

| Операция | Python | C# |
|-----------|---------|----|
| Вывод | `print()` | `Console.WriteLine()` |
| Ввод | `input()` | `Console.ReadLine()` |

**Различие:** Python имеет встроенные функции `print` и `input`, C# использует класс `Console` из стандартной библиотеки.

---

## 6. Пользовательские функции

| Элемент | Python | C# |
|----------|---------|----|
| Объявление | `def` | `returnType name(...)` |
| Возврат значения | `return` | `return` |
 
 Пример 
```python
def add(a, b):
    return a + b
```

```csharp
int Add(int a, int b) 
{
    return a + b;
}
```

**Различие:** Python не указывает типы явно, C# требует типизации параметров и возвращаемого значения.

---

## 7. Встроенные типы данных

| Тип | Python | C# |
|------|---------|----|
| Целое | `int` | `int`, `long`, `short`, `byte` |
| Вещественное | `float` | `float`, `double`, `decimal` |
| Строка | `str` | `string` |

**Различие:** Python динамически типизирован, C# — статически. В C# больше числовых подтипов.

## 9. Пользовательские составные типы данных

### 9.1. Массивы
| Элемент | Python | C# |
|----------|---------|----|
| Основная структура | `list` | `array` или `List<T>` |
| Объявление | `arr = [1, 2, 3]` | `int[] arr = {1, 2, 3};` |

### 9.2. Структуры 
| Элемент | Python | C# |
|----------|---------|----|
| Аналог | `class` или `dataclass` | `struct` |



**Различие:** В Python нет нативных структур, используются классы; в C# структуры — отдельный тип с семантикой значения.

---

## 10. Основные различия лексики Python и C#

1. **Синтаксис блоков:** Python использует отступы, C# — фигурные скобки.
2. **Типизация:** Python — динамическая, C# — статическая.
3. **Ключевые слова:** Python имеет компактный набор (`if`, `def`, `class`), C# — более формальный и типизированный (`public`, `int`, `struct`).
4. **Комментарии:** Python — `#`, C# — `//` и `/* ... */`.
5. **Выражения:** В Python выражения проще и короче, в C# — требуют явных типов и скобок.

---

## 11. Вывод
Python обладает минималистичной и гибкой лексической структурой, ориентированной на читаемость и простоту, тогда как C# отличается строгой и формализованной лексикой, направленной на безопасность типов и явную структуру программ.
